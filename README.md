# 📚 Books I Referred – ML | DSA | Python | Data Science

This repository contains a curated collection of books and notes that I have used while learning **Machine Learning, Data Science, Python, and Data Structures & Algorithms**.

---

## 🧠 Machine Learning

* **The Hundred-Page Machine Learning Book – Andriy Burkov**
  ➤ Type: Conceptual / Beginner-Friendly
  ➤ A concise introduction to core ML concepts, models, and practical intuition. 

* **Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow – Aurélien Géron**
  ➤ Type: Practical / Intermediate
  ➤ Covers real-world ML workflows, model building, and deep learning using modern tools. 

* **ML Ebook (Custom/Notes)**
  ➤ Type: Notes / Reference
  ➤ Personal or compiled learning material on machine learning topics.

---

## 📊 Statistics & Data Science

* **An Introduction to Statistical Learning (Python) – James, Witten, Hastie, Tibshirani**
  ➤ Type: Academic / Practical
  ➤ Covers statistical learning techniques with Python implementations. 

* **Python for Data Analysis – Wes McKinney**
  ➤ Type: Practical / Data Handling
  ➤ Focuses on data wrangling using pandas, NumPy, and Jupyter. 

* **Storytelling with Data – Cole Nussbaumer Knaflic**
  ➤ Type: Data Visualization / Business
  ➤ Teaches how to communicate insights effectively using visuals. 

---

## 💻 Data Structures & Algorithms

* **Data Structures and Algorithms Made Easy – Narasimha Karumanchi**
  ➤ Type: Interview Prep / Problem Solving
  ➤ Contains extensive problems with multiple solution approaches. 

* **Cracking the Coding Interview – Gayle Laakmann McDowell**
  ➤ Type: Interview Preparation
  ➤ 189 coding problems with detailed solutions and strategies. 

---

## 🐍 Python Programming

* **Python Crash Course – Eric Matthes**
  ➤ Type: Beginner / Project-Based
  ➤ Hands-on introduction to Python with real projects. 

* **Python Flash Cards – Eric Matthes**
  ➤ Type: Quick Revision
  ➤ Covers essential programming concepts and syntax in a concise format.
